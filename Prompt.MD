# AI Prompts Used During Development

This file lists some of the prompts used to get guidance from AI while building the **Cine-Stream Movie Explorer** project.

---

### Project Setup
- "How to create a React project using Vite and organize the folder structure for components, hooks, and context?"
- "Explain how to use React Router to create multiple pages in a React application."

---

### Fetching Movies from API
- "How to fetch data from the TMDB API in React and display movie posters and titles in a grid layout?"
- "How to handle API requests using useEffect and useState in React?"

---

### Search Feature
- "How to implement a search bar in React that fetches movie results from an API when the user types a movie name?"
- "How to implement debouncing in React so the API is not called on every keystroke?"

---

### Infinite Scroll
- "How to implement infinite scrolling in React to load more movies when the user reaches the bottom of the page?"
- "How to append new API results to the existing movie list in React?"

---

### Favorites Feature
- "How to implement a favorites feature in React where users can add or remove movies?"
- "How to store favorite movies in localStorage so they persist after refreshing the page?"
- "How to use React Context API to manage global state for favorite movies?"

### Movie Card Component
- "Create a reusable React MovieCard component with a poster, title, rating, and a favorite button."
- "How to conditionally render a filled heart icon when a movie is added to favorites?"

---

### Lazy Loading Images
- "How to use native lazy loading for images in React to improve performance?"

### Mood Matcher (AI Feature)
